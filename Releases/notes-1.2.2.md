## What's New in 1.2.2

- Sonic & Knuckles lock-on now works on Genesis — launch Sonic 1, 2, or 3 through Sonic & Knuckles for the combined cartridge experience

## Bug Fixes

- Fixed three crashes reported by users in production (import failure, session teardown, and a Core Data threading issue)
- The game window title now shows which core is running the current game
- Loading a save state from a different region or revision now prompts you to confirm instead of silently failing
- Dreamcast GDI disc images with unusual track layouts now import correctly
- Improved 3DO accuracy: MADAM matrix math now uses integer arithmetic, matching real hardware behavior
