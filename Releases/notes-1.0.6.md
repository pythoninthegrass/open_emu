## What's New in 1.0.6

- Flycast updated to version 2.4

## Bug Fixes

- Loading a save state created on an older core version now shows a warning instead of silently crashing
- Dreamcast BIOS files (dc_flash.bin) are now correctly recognised in System Files — all regional variants are accepted and duplicate entries are collapsed into a single row
- Core updates no longer stall or repeatedly offer the same version due to a URLSession cache issue
